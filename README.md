RESUMEN DE FUNCIONAMIENTO.

Al cargar la página, Python genera las 40 cartas y las mezcla.
El HTML las dibuja todas boca abajo.
Al hacer clic, el JavaScrip añade la clase .flipped que hace la animación de giro.
Si las dos cartas coinciden (Ej: Caballo y Caballo); se quedan abiertas y sumas un punto. Sigues jugando.
Si no coinciden, se giran solas tras 1 segundo y pasa el turno al otro jugador.

Para jugar una o dos personas.

Ayuda a relajar la mente y mantener el animo.
